Edit Keyframes
This section provides various editing options for modifying the graph layout.
To edit nodes and links, you must first select them—either by clicking individually or by drawing a selection rectangle.
You can expand the graph by adding new nodes and links.
Nodes can be moved using keyframes. When a node has a position keyframe in a frame, its position is fixed. Between keyframes, the position is interpolated. Keyframes can be dragged to new positions or deleted.
You can adjust the duration of links by setting their start and end frames, or by moving their duration keyframe block.
The same applies to a node’s health status. You can adjust each status keyframe block’s duration and add new blocks with the desired health state.
Finally, you can delete selected nodes and links.
